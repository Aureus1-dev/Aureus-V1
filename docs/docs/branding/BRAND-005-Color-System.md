# BRAND-005 — Color System

Status: Draft for Founder Review

---

# Purpose

This document establishes the official color philosophy and color system of Aureus.

Colors are selected to reinforce trust, stewardship, clarity, hope, and professionalism.

They are not selected according to temporary design trends.

---

# Color Philosophy

Every color used by Aureus should strengthen confidence in the institution.

The palette should feel:

- Timeless
- Calm
- Human
- Warm
- Intelligent
- Trustworthy
- Professional
- Accessible

---

# Emotional Objectives

The Aureus color system should communicate:

- Trust before excitement
- Clarity before attention
- Confidence before urgency
- Hope before fear
- Warmth before coldness
- Stability before novelty

---

# Primary Palette

The primary palette will ultimately consist of:

- Primary Brand Color
- Secondary Brand Color
- Accent Color
- Success Color
- Warning Color
- Error Color
- Information Color

Exact color values (HEX, RGB, CMYK, Pantone, and digital tokens) will be defined after Founder approval.

---

# Neutral Palette

A neutral palette shall provide:

- Background colors
- Surface colors
- Borders
- Dividers
- Typography support
- Interface structure

Neutrals should never overpower the primary brand identity.

---

# Accessibility

Every approved color combination must:

- Meet or exceed WCAG AA accessibility standards.

AAA compliance should be preferred whenever practical.

Color shall never be the sole indicator of meaning.

---

# Usage Principles

Colors should be applied consistently across:

- Website
- Mobile applications
- Documentation
- Social media
- Presentations
- Merchandise
- Printed materials
- Marketing
- Product interfaces

---

# Prohibited Practices

Avoid:

- Excessive gradients
- Neon palettes
- Low-contrast combinations
- Trend-driven color choices
- Inconsistent color usage

---

# Future Expansion

Future versions of this document will define:

- Official HEX values
- RGB values
- CMYK values
- Pantone references
- Design tokens
- Semantic UI colors
- Dark Mode palette
- Light Mode palette
- Accessibility ratios
- Component color mapping

---

# Canonical Principle

Color exists to strengthen understanding and trust.

It must never distract from the mission.

---

End of BRAND-005.
