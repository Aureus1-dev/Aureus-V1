# Aureus Branding Canon

Status: Canonical Structure (Founder Review)

## Purpose

The Branding Canon is the single source of truth for the visual identity of Aureus.

It defines how Aureus presents itself to the world through its logo, colors, typography, design language, website, applications, documents, and all other visual communication.

The Branding Canon exists to ensure that every design decision strengthens trust, clarity, professionalism, accessibility, timelessness, and the Aureus mission.

No branding decision becomes canonical until it has been approved and documented within this directory.

---

## Governance

Every branding document follows the lifecycle:

Draft → Founder Review → Approved → Canonical

Future modifications must be proposed through a Brand Amendment Proposal (BAP) before changing any approved canonical branding document.
